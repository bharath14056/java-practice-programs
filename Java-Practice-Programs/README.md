## ▶️ How to Run Programs

### Prerequisites
- Java JDK 8 or higher installed
- Terminal / Command Prompt

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/Java-Practice-Programs.git

# 2. Navigate to any folder
cd Java-Practice-Programs/02-Number-Programs

# 3. Compile a Java file
javac Factorial.java

# 4. Run the compiled class
java Factorial
```

### Using an IDE (Recommended)
- **IntelliJ IDEA** – Open the folder as a project, right-click any `.java` file → Run
- **VS Code** – Install the "Extension Pack for Java", open the file, press `F5`
- **Eclipse** – Import as Java project, run each file individually
