# 02 — Number Programs

Classic number theory programs commonly asked in Java interviews and placement rounds.

| Program | Concept |
|---------|---------|
| ArmstrongNumber | Power of digits sum = number |
| Factorial | Recursive-style factorial |
| FibonacciSeries | Series up to N terms |
| PalindromeCheck | Reverse comparison |
| GcdCalculator | Euclidean algorithm |
| StrongNumber | Sum of factorials of digits |
| DisariumNumber | Positional power sum |
| DuckNumber | Contains 0 but not at start |
